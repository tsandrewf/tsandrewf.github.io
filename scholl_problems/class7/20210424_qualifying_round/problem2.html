Задача 2. Настя выписала на доску все трехзначные числа, для которых средняя цифра равна сумме сумме двух остальных цифр, уменьшенной на один. Сколько чисел выписала Настя?
Решение. Для каждой цифры \(n\) от 0 до 8 Настя написала \((n + 1)\) чисел. Например, для цифры "0" написала 1 число (100), для цифры "1" написала 2 числа (210 и 111), для цифры "2" - 3 числа (320, 221 и 122), для цифры "3" - 4 числа (430, 331, 232 и 133) и т.д. Итого, количество написанных Настей чисел равно сумме арифметическо прогрессии: 1, 2, 3, ..., 10. А вот для цифры "9" Настя смогла написать не 10, только 9 чисел (991, 892, 793, 694, 595, 496, 397, 298 и 199). \[(\sum_{i=1}^{9}i)+9={{(1+9) \times 9} \over 2}+9=45\]
Ответ: 45
