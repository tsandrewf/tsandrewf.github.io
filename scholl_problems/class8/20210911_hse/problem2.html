Вопрос 2. Начав с натурального числа \(N\), несколько раз проводят следующую операцию. За одну операцию имеющееся число удваивают, а затем уменьшают на 6400. После нескольких таких операций получили 0. При каком наибольшем \(N\) такое могло быть?
Решение. Проведем операцию в обратном направлении, начиная с конечного числа 0, до тех пор, пока не наткнемся на ненатуральное число. Последнее натуральное число и будет искомым решением. \[{(0 + 6400) \over 2} = 3200\] \[{(3200 + 6400) \over 2} = 4800\] \[{(4800 + 6400) \over 2} = 5600\] \[{(5600 + 6400) \over 2} = 6000\] \[{(6000 + 6400) \over 2} = 6200\] \[{(6200 + 6400) \over 2} = 6300\] \[{(6300 + 6400) \over 2} = 6350\] \[{(6350 + 6400) \over 2} = \color{red}{6375}\] \[{(6375 + 6400) \over 2} = \xcancel{6387,5}\]
Ответ: 6375
