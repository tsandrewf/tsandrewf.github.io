Вопрос 4. Правильная дробь называется аликвотной, если ее числитель равен 1. Сколько существует пар различных аликвотных дробей (порядок в паре не важен), в каждой из которых сумма будет аликвотной дробью, большей \(1 \over 6\)?
Решение. Сумма двух аликвотных дробей является аликвотной только в том случае, если обе дроби аликвотные и их знаменатель кратен 2. Поэтому нужно просто перечислить все варианты, начиная с \(1 \over 2\), до тех пор, пока сумма превышает \(1 \over 6\). \[{1 \over 2} + {1 \over 2} = 1 \gt {1 \over 6}\] \[{1 \over 4} + {1 \over 4} = {1 \over 2} \gt {1 \over 6}\] \[{1 \over 6} + {1 \over 6} = {1 \over 3} \gt {1 \over 6}\] \[{1 \over 8} + {1 \over 8} = {1 \over 4} \gt {1 \over 6}\] \[{1 \over 10} + {1 \over 10} = {1 \over 5} \gt {1 \over 6}\] \[\xcancel{{1 \over 12} + {1 \over 12} = {1 \over 6} = {1 \over 6}}\]
Ответ: 5
